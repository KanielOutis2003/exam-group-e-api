# Exam Group E API

A Node.js REST API for the exam project.

## Description
This API provides endpoints for managing data for the Exam Group E project.

## Installation
```
npm install
```

## Running the Application
```
npm start
```

## Development
```
npm run dev
```

## Testing
```
npm test
```
